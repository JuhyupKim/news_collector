# � 뉴스 자동 수집 및 분류 크롤러 프로젝트 (News Scraper Suite)

환영합니다! 이 프로젝트는 여러 IT, 가전, 기술 매체의 최신 뉴스를 자동으로 수집하고 하나의 깔끔한 **엑셀(CSV) 파일**로 정리해주는 프로그램입니다. 파이썬(Python)을 처음 다루시는 분들도 이 가이드를 따라 쉽게 실행하실 수 있도록 구성했습니다.

---

## 📌 주요 기능
- **🚀 초고속 병렬 수집 (멀티스레딩 활용)**
  5개의 뉴스 사이트(ZDWang, 삼성 뉴스룸, CHEAA, 테크월드뉴스, 로봇신문)에서 동시에 뉴스를 수집하여 작업 시간을 크게 단축합니다.
- **🏷️ 스마트 카테고리 자동 분류**
  기사에 포함된 태그, 제목 기호, 페이지 경로 등을 인공지능처럼 영리하게 분석하여 메인 카테고리와 서브 카테고리를 알아서 분류합니다.
- **🕒 날짜 자동 정리**
  웹사이트마다 제각각인 날짜 표기법(예: `2024.10.12`, `10/12/2024`)을 하나의 표준 형식(`2024-10-12 14:00:00`)으로 통일합니다.
- **🧠 스마트 중복 방지 (Cross-Run Deduplication)**
  매일 크롤러를 실행해도 이전에 수집한 뉴스를 기억하고 건너뜁니다! (수집 속도 향상 및 중복 데이터 방지)
- **⚡ 수집량 제어 (Limit Control)**
  각 뉴스 사이트별로 수집할 최대 기사 개수를 지정하여 시스템 과부하를 막고 핵심 뉴스만 빠르게 수집할 수 있습니다.

## 🚀 초보자 가이드: 프로그램 실행 방법

### 1단계: 파이썬(Python) 준비하기
이 프로그램을 실행하려면 컴퓨터에 파이썬이 설치되어 있어야 합니다.
- 파이썬이 없다면 [Python 공식 홈페이지](https://www.python.org/downloads/)에서 최신 버전을 다운로드하여 설치해 주세요.
- **중요**: 파이썬 설치 화면에서 하단의 `Add Python to PATH`를 **반드시 체크**하고 설치해야 합니다!

### 2단계: 필수 준비물(라이브러리) 설치하기
프로그램이 뉴스를 수집하고, 중국어를 번역하고, 브라우저를 조종하려면 특별한 도구들이 필요합니다.
1. 키보드의 `Windows 로고 키` + `R` 키를 누르고 `cmd`를 입력 후 엔터(Enter)를 누르세요. 검은색 화면(명령 프롬프트)이 켜집니다.
2. 아래의 명령어를 복사해서 검은색 화면에 붙여넣기 한 뒤 엔터를 누르세요.
   ```bash
   pip install requests beautifulsoup4 pandas selenium webdriver-manager deep-translator
   ```
3. 설치가 완료될 때까지 잠시 기다립니다.

### 3단계: 뉴스 수집 시작!
모든 준비가 끝났습니다. 이제 진짜 프로그램을 실행해 볼 차례입니다.

1. 열려있는 명령 프롬프트(cmd) 창에서, 이 프로그램이 저장된 폴더로 이동해야 합니다. 
   예를 들어, 폴더가 `D:\git\newsletter`에 있다면 다음 명령어를 입력하고 엔터를 누르세요.
   ```bash
   cd /d D:\git\newsletter
   ```
2. 이제 메인 프로그램을 실행합니다.
   ```bash
   python news_collector.py
   ```
3. 화면에 **"🚀 최근 1일치 데이터 수집을 시작합니다..."** 라는 메시지가 나오면서 수집이 시작됩니다. 
   수집이 끝나면 `output/` 폴더 안에 오늘 날짜와 시간(예: `260305_120000_competitor.csv`)이 적힌 엑셀 파일이 짠! 하고 만들어집니다. 그것을 열어보시면 됩니다.

## ⚙️ 심화 안내 (설정 변경 방법)
만약 **"어제 뉴스말고 최근 3일 치 뉴스를 뽑고 싶어!"** 라고 하신다면, `news_collector.py` 파일을 메모장이나 파이썬 편집기(IDE)로 열고 18번째 줄을 수정해 주세요.

```python
# 수정 전
DATE_THRESHOLD = 1 
MAX_ITEMS_PER_CRAWLER = 100

# 수정 후 (최근 3일 치 수집, 수집 개수 제한 해제)
DATE_THRESHOLD = 3 
MAX_ITEMS_PER_CRAWLER = 0 # 0 또는 None으로 설정하면 개수 제한 없이 모두 수집합니다.
```
숫자를 원하는 만큼 바꾸고 저장한 뒤 다시 프로그램을 실행하면 됩니다.

**💡 꿀팁: 수집 기록 초기화하기**  
프로그램이 이전 수집 기록을 잊고 처음부터 다시 수집하게 만들고 싶다면, 생성된 `output/.crawled_history.txt` 파일을 삭제하신 후 실행하시면 됩니다!

## 🛠️ 내가 원하는 뉴스 사이트 추가하기 (초보자용 크롤러 마법사)
새로운 뉴스 사이트를 추가하고 싶으신가요? HTML이나 파이썬을 전혀 몰라도 컴퓨터와 대화만 하면 알아서 뼈대 코드를 만들어주는 마법사를 준비했습니다!

1. 명령 프롬프트(cmd) 창에 아래 명령어를 입력하세요:
   ```bash
   python easy_crawler_maker.py
   ```
2. 화면의 질문에 따라 **영문 이름**(예: `hankook`), **한글 이름**(예: `한국일보`), 그리고 뉴스가 있는 **웹사이트 주소**(예: `https://www.hankookilbo.com/News/List`)를 차례대로 입력합니다.
3. 컴퓨터가 잠시 해당 사이트를 분석한 뒤, 화면에 발견된 글자들을 보여주며 질문합니다.
   - 💻 *"👉 발견 문구: [삼성전자 스마트폰 출시...] 이 문구가 기사 제목이 맞나요? (y/n)"*
   - 😎 사용자는 맞으면 **y**, 틀리면 **n**만 눌러주시면 됩니다!
4. 제목, 날짜, 상세 본문 위치를 모두 찾으면 자동으로 `hankook_crawler.py` 파일이 완성됩니다.
5. 이제 완성된 파일을 `news_collector.py`에 등록해주기만 하면 끝입니다! (등록 방법은 마법사가 마지막에 화면에 자세히 알려줍니다)

## 📝 크롤러별 추출 특징 요약
궁금하신 분들을 위해, 각 사이트별로 어떻게 똑똑하게 뉴스를 뽑아내는지 설명합니다.
- **ZDWang**: 홈페이지 자체의 카테고리 맵을 읽어서 원래 분류대로 가져옵니다.
- **Samsung**: 투명망토를 입은 브라우저(Selenium)가 직접 접속하여 숨겨진 메타데이터(`<json-ld>`)를 해석해 분류합니다.
- **Techworld / IRobotNews**: 제목(`title`) 안에 있는 `<` 기호를 기준으로 분류를 정밀하게 쪼개어 가져옵니다.
- **CHEAA**: 기사 본문이나 제목 속에 숨겨진 `-`(하이픈) 기호를 기반으로 카테고리를 알아서 나눕니다.
